## Goal

Prevent D-Bus model reorder from switching the island/ControlCenter active player. Keep Controls.qml as the single media owner.

## Exact allowlist

- `tahoe-shell/services/Controls.qml`
- `tahoe-shell/services/controls/MediaPlayerSelection.js`
- `tahoe-shell/tests/test_controls_mpris_selection.py`
- `tahoe-shell/docs/dynamic-island-v2-task-reports/T04.md`

## Implementation

- Pure `MediaPlayerSelection.js` with sticky dbusName memory rules
- `lastActivePlayerDbusName` + `onActivePlayerChanged` update
- Playing preempts paused; multi-playing uses remembered-if-playing else lex dbusName
- Ineligible (no title, no control capability) never selected
