## Forbidden

- Second notification model or island-owned live ID queue for production arbitration
- dbus-monitor
- Multiple restore/drain presenters racing on stateChanged
- Changing public IPC names

## Implementation

- Priority helpers in `DynamicIslandReducer.js` (`blocksOsd` / `blocksWorkspace` / `blocksNotification`)
- `blocksTransientOsd` / `blocksTransientWorkspace` yield via arbitration flags (active notification lease)
- Live island ID queue removed; `completedNotificationIds` + `nextPresentableLiveNotification()` resolve Notifications FIFO
- Manual IPC payloads remain on `pendingNotificationIds`
- Single `restoreAfterTransient()` drain; reducer apply suppresses re-entrant onStateChanged restore
- Abort paths (`SHOW_*` / expand / hover / collapse) emit `endNotificationLease` (mark completed + clear id)
- Notifications: `findActiveById` + `presentationHead` minimal lease queries
- T02 priority xfails removed (GREEN)
