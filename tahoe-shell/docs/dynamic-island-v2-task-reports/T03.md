## Implementation

- Pure reducers: `applyWorkspaceActivated`, `applyWorkspaceUrgencyChanged`, `applyWorkspaceActiveWindowChanged`
- Queries: `focusedOutputName`, `activeWorkspaceForOutput`, `activeWorkspaceIndexForOutput`
- `activeWorkspace` prefers IPC focused workspace; falls back to WindowManager
- `workspaceLabel` uses name → idx (not stable id)
- Reconnect/exit clears IPC workspace baseline before next `WorkspacesChanged`
- Hotplug cleanup primarily via niri full `WorkspacesChanged`; optional `pruneStaleOutputWorkspaces`
