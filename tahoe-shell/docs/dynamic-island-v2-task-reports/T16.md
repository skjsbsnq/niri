## Goal

- True top-bar compact media: 22×22 art, single-line title, play/pause TahoeSymbol.
- Optional 2px progress when Controls supports position.
- No artist, no fake visualizer, no MPRIS re-selection.
- Content-driven width 200–224; TopBar reserve remains max compact media.
- Smooth clock↔media morph (shared compact layer opacity/y).
- mediaTrackTitle from Controls only; exit latch on player disappear.

## Implementation summary

1. DynamicIslandCompactMediaView: art (visibility-gated load), title elide, play/pause symbol, optional progress overlay (no row reflow).
2. Content replaces plain compactLabel with CompactMediaView; shares compactLayerOpacity/Y with clock; latches title/width on exit.
3. Overlay measures compactMediaContentWidth and clamps to v2CompactMediaWidthMin/Max; Theme.accent + progress track; mediaTrackTitle from service Controls mirror.
4. Service exposes mediaTrackTitle from controlsService.trackTitle (never displayText/clock).
5. reduced motion uses v2ReducedContentMs for compact layer Behaviors.
