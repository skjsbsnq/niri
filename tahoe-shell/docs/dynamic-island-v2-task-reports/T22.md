## Implementation

- Removed the retired summary properties, generic detail row, unused bottom
  progress track, dead timer visibility forwarding, and superseded motion
  aliases/tokens.
- Kept expanded media under its existing Loader and moved the notification
  scene (including its Flickable and Images) under a current-state Loader.
- Removed the unused Dynamic Island battery service wiring and other dead
  service/overlay properties without changing the public IPC compatibility
  functions.
- Hardened AppMenu's single probe pipeline after runtime profiling showed that
  stable `{windowId,pid,appId}` wrapper replacement repeatedly spawned
  `appmenu_probe.py`. Normal same-identity refreshes now reuse the applied
  result; explicit menu/recovery demand still refreshes.
