## Goal

- When the island is enabled, only the Overlay renders the resting clock.
- When disabled, TopBar renders ordinary readable time text (not a faux island chip).
- Remove the dual visual path (`DynamicIslandChip` vs Overlay).
- V2 clock: localized weekday + 24h time; content-driven width 112–136; height 32.
- TopBar center reserve covers max compact media width and does not thrash on state switches.
