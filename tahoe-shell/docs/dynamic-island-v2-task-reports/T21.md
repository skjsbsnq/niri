# T21: Surface shared Bluetooth events

Status: review
Base SHA: `72d0924bc1c30a5f2cc00fcb1fe6959ea8843ff3`

## Goal

- Surface user-valuable Bluetooth connecting, connected, failed, and disconnected events.
- Keep `Controls.qml` as the only Bluetooth owner and emit immutable event snapshots.
- Coalesce consecutive events for the same device.
- Preserve notification priority and event-output ownership.
- Keep Control Center and Settings Bluetooth behavior unchanged.

## Prohibited

- No `bluetoothctl` polling loop.
- No second Bluetooth tracker or backend.
- No copied Control Center device list.
- No live Bluetooth device object retained by Dynamic Island.
- No broadcast of the same transient to every output.
