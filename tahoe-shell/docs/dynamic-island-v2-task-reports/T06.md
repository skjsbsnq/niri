## Goal

Extract a pure, testable presentation reducer from DynamicIsland.qml without creating a second production state machine. Migrate uncontroversial base decisions only: clock, media availability, manual expand/collapse. Public properties and IPC remain compatible.

## Exact allowlist

- `tahoe-shell/services/DynamicIsland.qml`
- `tahoe-shell/services/DynamicIslandReducer.js`
- `tahoe-shell/tests/test_dynamic_island_reducer.py`
- `tahoe-shell/docs/dynamic-island-v2-task-reports/T06.md`

## Forbidden

- Second DynamicIsland service or parallel orchestrator
- Dual old/new decision paths for migrated judgments
- User-visible V2 feature toggle
- Accessing QML services, Date.now, or Timers from the reducer
- Changing public IPC names or property semantics
- Migrating transient arbitration, output owner, or swipe settle (T07–T09)

## Implementation

- Pure `DynamicIslandReducer.js`: state slice `{forcedState, preferMediaWhenAvailable, hoverExpanded}`, context, events, `reduce()` → `{state, effects}`
- Migrated production decisions: resting/normalized presentation, SHOW_TIME/MEDIA/EXPANDED_*, TOGGLE_EXPANDED, COLLAPSE, MEDIA_AVAILABILITY_CHANGED, HOVER_EXPAND/COLLAPSE, RESET, ISLAND_DISABLED
- `DynamicIsland.qml` is sole orchestrator: `dispatchPresentation` applies slice + runs effects
- **Apply order:** cleanup effects (stop timer, clear fields/queues/swipe) run before `forcedState` commit; drain effects run after (fixes onStateChanged re-entrancy)
- **ISLAND_DISABLED** preserves `preferMediaWhenAvailable`; only RESET resets it
- Disable path keeps source-visible `clearPendingNotificationIds()` for notification identity contract
- Transient arbitration / output pin / swipe settle intentionally untouched (T07–T09)
